# de.dennis-kraffczyk.wsc.usercardemail

This repository contains the source code of the `de.dennis-kraffczyk.wsc.usercardemail` - plugin for [WoltLab Suite Core](https://github.com/WoltLab/WCF). 

The package itself injects the email address of each user in their user card and is therefore display everywhere, where the user card is used.

## Installation
### Manual
1. Download the `de.dennis-kraffczyk.wsc.usercardemail.tar.gz` file from the [latest elease](https://github.com/Fighter456/de.dennis-kraffczyk.wsc.usercardemail/releases/latest).
2. Open the WoltLab Suite Administration Panel and navigate to `Configuration --> Packages`
3. Click on `Install package` and then on `Upload package`
4. Select the previously download file (from step 1), submit the dialog and follow the instructions on the screen

## System Requirements
The package is only available for WoltLab Suite 6.2+

## Required Permission
To view the email address in the user card it is required that your user account has the permission named `Can view private user options`.
